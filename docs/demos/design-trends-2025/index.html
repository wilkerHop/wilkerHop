Key Findings

  • ➤
    The "Bento" Takeover:

    Driven by Apple and Linear, the grid-based, content-first layout has become the standard for SaaS, replacing complex hero sections.

  • ➤
    Softening Edges:

    While Neo-Brutalism persists in niche creative brands, mainstream design is moving toward "Aurora" (Glass 2.0) for a friendlier feel.

  • ➤
    The Eco-Imperative:

    Low-energy web design is influencing aesthetics, leading to darker modes by default and vector-heavy visuals over raster images.
